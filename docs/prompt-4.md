The latest project involves building a Microsoft Excel equivalent for use by AI agents.

The productivity of most office workers revolves around:

1) Gather information and data. Then tabulate them using Excel. More advanced use may involve creating pivot tables and charts.

2) Writing reports using Word.

3) Creating presentation slides for meetings using PowerPoint.

To achieve this, we need to create AI-friendly replacements for Microsoft Office applications. Traditional software sells based on how friendly it is to humans. Unfortunately, what is friendly to humans is very unfriendly to AI.

If you give AI Command Line Interface access to an API-based application, it's like giving water to fish.
